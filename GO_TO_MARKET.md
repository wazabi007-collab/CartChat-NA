# OshiCart — Go-To-Market Plan

## First 10 Pilot Merchants

### Selection Criteria
- Already selling on WhatsApp (proven demand)
- 10+ products to catalog
- Processes 5+ orders/week (enough volume to feel pain)
- Based in Windhoek (for in-person onboarding if needed)
- Willing to give feedback weekly

### Target Segments (2 merchants each)
1. **Fashion resellers** — Instagram + WhatsApp sellers, high product turnover
2. **Home bakers / food sellers** — Daily order volume, time-sensitive products
3. **Beauty / cosmetics** — High repeat purchase, price-sensitive customers
4. **Phone accessories / electronics** — Catalog-heavy, price comparison shoppers
5. **Farm produce / local goods** — Emerging market, community-driven sales

### Sourcing Channels
- WhatsApp Status sellers (search Windhoek buy-and-sell groups)
- Instagram business accounts with "DM to order" or "WhatsApp to order" in bio
- Facebook Marketplace Namibia sellers
- Personal network referrals
- Namibian SME WhatsApp groups

## Onboarding Script

### Initial Outreach (WhatsApp Message)
```
Hi [Name]! I saw you sell [products] on WhatsApp — nice stuff!

I'm building a free tool called OshiCart that lets you:
- Create a product catalog link you can share on WhatsApp
- Customers browse and order directly (no more back-and-forth messages)
- See all orders + proof of payment in one dashboard

It's free for up to 20 products. Takes 5 minutes to set up.

Would you like to try it? I'll help you set up personally.
```

### Onboarding Call (5-10 min)
1. **Screen share or sit together** — walk through sign up
2. **Add 5 products together** — show how quick it is
3. **Generate store link** — have them share it in one WhatsApp group immediately
4. **Show order dashboard** — explain the flow
5. **Set expectation**: "I'll check in with you in 3 days to hear how it's going"

### Follow-Up Cadence
- **Day 3**: "How many orders came through? Any issues?"
- **Day 7**: "What would make this better for you?"
- **Day 14**: Formal feedback session (5 questions, voice note OK)
- **Day 21**: Ask for referral to one other seller
- **Day 30**: Discuss Pro tier if they've hit limits or want more features

## Pricing Draft (NAD)

| Tier | Price | Includes |
|------|-------|----------|
| **Free** | NAD 0 | 20 products, 50 orders/month, basic storefront, manual payment proof |
| **Pro** | NAD 99/month | Unlimited products, unlimited orders, analytics, priority support |
| **Business** | NAD 249/month | Pro + 3 staff accounts, custom branding, featured in directory |

### Pricing Rationale
- **NAD 99 = ~$5.50 USD** — affordable for a merchant doing NAD 20,000+/month revenue
- **ROI pitch**: "If OshiCart saves you 2 hours/day on order management, that's 60 hours/month. Is that worth NAD 99?"
- **Comparison**: A single WhatsApp Business API conversation costs ~NAD 2-4. At 50 orders/month, API alone would cost more than Pro tier
- **Collection method (V1)**: Manual EFT to OshiCart bank account. Monthly. Admin tracks in spreadsheet until automated billing in V2

### Upgrade Triggers
- Hit 20 product limit → prompt to upgrade
- Hit 50 order limit → prompt to upgrade
- Request analytics → show preview, upgrade to unlock
- Request staff accounts → Business tier prompt

## Sales Channels

### Channel 1: Direct WhatsApp Outreach (Week 1-4)
- Target: 50 outreach messages → 15 responses → 10 signups → 5 active
- Source from WhatsApp groups, Instagram, Facebook Marketplace
- Personal, 1:1 messages (not broadcast)

### Channel 2: WhatsApp Status Marketing (Week 2+)
- Post store creation tutorials on own WhatsApp Status
- Show merchant success stories (with permission)
- "How [Merchant] got 20 orders in one week using OshiCart"

### Channel 3: Facebook Groups (Week 3+)
- Join Namibian entrepreneur/SME groups
- Share value-first content: "5 tips to sell more on WhatsApp"
- Soft CTA: "I built a free tool that does #3 automatically"

### Channel 4: Referral Program (Week 4+)
- "Refer a seller → both get 1 month Pro free"
- WhatsApp-native sharing (pre-filled referral message)

### Channel 5: Instagram Reels (Week 4+)
- 30-second demos: "Set up your WhatsApp store in 60 seconds"
- Before/after: manual WhatsApp selling vs OshiCart

## Weekly Execution Cadence

### Monday: Plan
- Review last week's metrics (signups, active merchants, orders processed)
- Set 3 priorities for the week
- Plan outreach targets

### Tuesday-Thursday: Execute
- Product development (features, bugs, polish)
- Merchant outreach (5 new contacts/day)
- Support existing merchants (respond within 2 hours)

### Friday: Ship + Measure
- Deploy weekly update
- Send "What's New" WhatsApp message to merchants
- Update metrics dashboard
- Write 1 social media post about OshiCart

### Saturday: Learn
- Review merchant feedback
- Analyze drop-off points (where do merchants or customers get stuck?)
- Update TASKS.md with learnings
- Plan next week's priorities

### Sunday: Rest or content creation
