# OshiCart Video Scripts & Runway Prompts

## How to Use This Guide
1. Use the **Runway Prompts** to generate video clips in Runway Gen-3/Gen-4
2. Use the **Video Scripts** as voiceover text (record yourself or use AI voice tools like ElevenLabs)
3. Combine clips + voiceover + screen recordings in **CapCut** or **DaVinci Resolve**
4. Export for each platform (9:16 for TikTok/Reels/Stories, 16:9 for YouTube/Facebook)

## Production Workflow

### For Each Video:
1. **Write script** (use the scripts above)
2. **Generate Runway clips** (use the prompts above, 3-4 seconds each)
3. **Screen record** the product (OBS Studio on desktop, or phone screen record)
4. **Record voiceover** (your voice or ElevenLabs AI)
5. **Edit in CapCut:**
   - Import Runway clips + screen recordings
   - Add voiceover audio
   - Add text overlays and transitions
   - Add background music (CapCut has free music)
   - Add OshiCart logo watermark
6. **Export:**
   - 9:16 (1080x1920) for TikTok, Reels, Stories
   - 16:9 (1920x1080) for YouTube, Facebook feed
   - 1:1 (1080x1080) for Instagram feed
7. **Post** using Meta Business Suite (schedule across Facebook + Instagram)

### Runway Tips:
- Use "Gen-3 Alpha Turbo" for faster generation
- Generate 3-4 variations of each prompt and pick the best
- Keep clips 3-5 seconds (stitch them in CapCut)
- Add "cinematic, 4K, shallow depth of field" to all prompts for professional look
- Use "African" in prompts to get representative visuals
- Avoid text in Runway-generated clips (add text in CapCut instead)
